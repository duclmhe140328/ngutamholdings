# v34 fixed – Upload ảnh từ máy + link URL

Bản này sửa lỗi bộ cài v34 dừng ở route `/api/health`, nên UI chưa được vá và chưa xuất hiện nút chọn ảnh.

Áp dụng cho:
- Ảnh sản phẩm: link + chọn nhiều ảnh trong máy.
- Logo shop: link + chọn ảnh trong máy.
- Banner shop: link + chọn ảnh trong máy.
- 3 ảnh slider shop: link + chọn ảnh trong máy.

Chạy từ thư mục gốc project:

```powershell
node .\saas-device-image-upload-cloud-v34-fixed\install-device-image-upload-v34-fixed.cjs
```
