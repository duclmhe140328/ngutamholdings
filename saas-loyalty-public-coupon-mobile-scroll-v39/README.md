# v39 — Cuộn phần ưu đãi công khai trên mobile

Sửa phần `MÃ ĐANG ÁP DỤNG / Ưu đãi công khai` trong ví xu của trang shop:

- Không còn bị giữ cố định ở đáy panel trên điện thoại.
- Toàn bộ panel cuộn chung; header vẫn bám trên.
- Chừa khoảng trống cuối nội dung cho safe-area, giỏ hàng và nút chat.
- Không đổi API, coupon, vòng quay, OTP hoặc dữ liệu.

## Cài đặt

Chạy tại thư mục gốc project:

```powershell
node .\saas-loyalty-public-coupon-mobile-scroll-v39\install-loyalty-coupon-scroll-v39.cjs
```
